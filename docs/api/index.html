Product Endpoints

GET
/product

Returns list of product document subsets, including, at least, the product id and name. These could be used to display thumbnail information about a business' products or to operate on a specific product using the provided id.

Query Params

?full - Flag. Returns full product documents.
?limit={number} - Limits the result count.
?ids={csv:string} - Comma delimited product ids. Returns list of products according to supplied ids.

/product/<id>
